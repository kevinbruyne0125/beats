Installing BEATS

Run the following from the command line:

sudo gem install beats

This will download BEATS from rubygems.org and add a beats command to your path.

The current version of BEATS is 1.2.0. If you have an older version, you can update with this:

sudo gem update beats

Not sure what version you have? Run beats -v.

Example Sounds

BEATS is not terribly useful unless you have drum sounds to use with it. To get you started, here are 20 high-quality drum sounds sampled from a Casio SA-20 keyboard that the author got when he was 8 years old: casio_sa20_drum_sounds.zip
