Installing BEATS

From the command-line, run the following to install BEATS:

sudo gem install beats

This will add a beats command to your path.

The current version of BEATS is 1.1.0. If you have version 1.0.0, you can update with this command:

sudo gem update beats

(You can run beats -v to determine your version.)
